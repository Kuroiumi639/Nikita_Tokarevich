Блок навигации

  • Ширина 15em
  • Должен присуствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присуствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседней равным .2em, при помощи свойства margin-botton

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присуствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседней равным .2em, при помощи свойства margin-botton. Картинка вставлена при помощи тега, плавание организовано по правомукраю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определятся помещённым внутрь контентом.
